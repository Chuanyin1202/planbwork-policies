Privacy Policy

Last Updated: September 2, 2025

Planbwork (hereinafter referred to as "we") highly values your privacy rights. This privacy policy explains how the Voxly app (hereinafter referred to as "this app") collects, uses, stores, and protects your personal data. By using this app, you agree to the terms of this privacy policy.

1. Data Collection Scope

1.1 Data You Actively Provide

  • Mood record content: Diary content you input through voice or text
  • Audio recordings: Voice content you record (only used for text transcription)
  • Community interaction content: Anonymous content you post in Tree Hole and Message in a Bottle features

1.2 Automatically Collected Data

  • Device identifier: Used to generate anonymous user ID
  • Usage statistics: App usage frequency, feature usage (collected anonymously)
  • Error reports: Technical information from app crashes or errors
  • Location information: Only when you choose to add location tags during recording (optional feature)

Important Note: We do not collect your real name, email address, phone number, or other personally identifiable information.

2. Data Usage Methods

Data Type Usage Purpose Processing Method
Mood Records Personal diary storage and viewing Local SQLite database storage
Audio Files Voice-to-text transcription Processed via OpenAI Whisper API then deleted
Record Text Emotion analysis, text optimization Analyzed via OpenAI GPT-4o-mini
Community Content Tree Hole, Message in a Bottle features Anonymous storage in Firebase Firestore
Usage Statistics Improve app experience Anonymous analysis via Firebase Analytics

3. Data Storage and Security

3.1 Local Storage

  • Personal mood records are stored in your device's SQLite database
  • Supports biometric (fingerprint/face) encryption protection
  • Data is not automatically backed up to the cloud

3.2 Cloud Processing

OpenAI API:

  • Audio files are only used for transcription and immediately deleted after processing
  • Text content is used for emotion analysis but not permanently stored
  • Complies with OpenAI's data processing policies

Firebase Services:

  • Community features use anonymous ID system
  • Uses Google Cloud security infrastructure

3.3 Security Measures

  • All network transmissions use HTTPS encryption
  • API keys are securely stored and not exposed on client side
  • Regular security patch updates

4. Data Sharing Policy

We do not sell, trade, or otherwise transfer your personal data to third parties, except in the following circumstances:

4.1 Service Providers

  • OpenAI - Voice-to-text and emotion analysis services
  • Google Firebase - Infrastructure and analytics services

4.2 Legal Requirements

When required by law or judicial process

4.3 Rights Protection

To protect our or others' rights, property, or safety

5. Your Rights

5.1 Data Control Rights

  • View right: View all your record content at any time
  • Modification right: Edit or delete any record
  • Export right: Export records as files for backup
  • Deletion right: Completely delete all data within the app

5.2 Privacy Settings

  • Choose whether to use biometric protection
  • Control whether to record location information
  • Choose whether to participate in community features
  • Manage notification reminder settings

6. Children's Privacy Protection

This app is not designed for children under 13 years of age. We do not knowingly collect personal data from children under 13. If you discover that a child has provided us with personal data, please contact us immediately, and we will take steps to delete the relevant data.

7. Data Retention Period

Data Type Retention Period Description
Personal Records Permanent (until you delete) Stored locally on device
Audio Files Immediately deleted after processing Not retained in cloud
Tree Hole Posts Limited period According to system operational needs
Message in a Bottle Limited period Cleaned up after conversation ends
Analytics Data According to service needs Anonymous statistical use

8. Cookies and Tracking Technologies

This app does not use cookies. We use the following technologies to improve services:

  • Firebase Analytics: Anonymous collection of usage statistics
  • Firebase Crashlytics: Collection of crash reports
  • Local Storage: Store app settings and preferences
