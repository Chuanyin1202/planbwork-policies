隱私政策

最後更新日期：2025 年 9 月 2 日

Planbwork（以下簡稱「我們」）非常重視您的隱私權。本隱私政策說明 Voxly 應用程式（以下簡稱「本應用」）如何收集、使用、儲存和保護您的個人資料。使用本應用即表示您同意本隱私政策的條款。

1. 資料收集範圍

1.1 您主動提供的資料

  • 心情記錄內容：您透過語音或文字輸入的日記內容
  • 錄音檔案：您錄製的語音內容（僅用於轉文字處理）
  • 社群互動內容：您在小樹洞、漂流瓶發布的匿名內容

1.2 自動收集的資料

  • 設備識別碼：用於生成匿名用戶 ID
  • 使用統計資料：應用使用頻率、功能使用情況（匿名收集）
  • 錯誤報告：應用崩潰或錯誤時的技術資訊
  • 位置資訊：僅在您記錄時選擇加入位置標記（可選功能）

重要說明：我們不會收集您的真實姓名、電子郵件、電話號碼或其他可直接識別身份的個人資訊。

2. 資料使用方式

資料類型 使用目的 處理方式
心情記錄 個人日記儲存與檢視 本地 SQLite 資料庫儲存
錄音檔案 語音轉文字 透過 OpenAI Whisper API 處理後刪除
記錄文字 情緒分析、文字優化 透過 OpenAI GPT-4o-mini 分析
社群內容 小樹洞、漂流瓶功能 Firebase Firestore 匿名儲存
使用統計 改善應用體驗 Firebase Analytics 匿名分析

3. 資料儲存與安全

3.1 本地儲存

  • 個人心情記錄儲存在您設備的 SQLite 資料庫中
  • 支援生物識別（指紋/臉部）加密保護
  • 資料不會自動備份到雲端

3.2 雲端處理

OpenAI API：

  • 語音檔案僅用於轉錄，處理後立即刪除
  • 文字內容用於情緒分析，不會永久儲存
  • 遵循 OpenAI 的資料處理政策

Firebase 服務：

  • 社群功能使用匿名 ID 系統
  • 採用 Google Cloud 安全基礎設施

3.3 安全措施

  • 所有網路傳輸採用 HTTPS 加密
  • API 金鑰安全儲存，不會暴露在客戶端
  • 定期更新安全修補程式

4. 資料分享政策

我們不會出售、交易或以其他方式向第三方傳輸您的個人資料，除以下情況：

4.1 服務提供商

  • OpenAI - 語音轉文字和情緒分析服務
  • Google Firebase - 基礎設施和分析服務

4.2 法律要求

應法律要求或司法程序需要時

4.3 保護權益

保護我們或他人的權利、財產或安全時

5. 您的權利

5.1 資料控制權

  • 查看權：隨時查看您的所有記錄內容
  • 修改權：編輯或刪除任何記錄
  • 匯出權：將記錄匯出為檔案備份
  • 刪除權：完全刪除應用內的所有資料

5.2 隱私設定

  • 選擇是否使用生物識別保護
  • 控制是否記錄位置資訊
  • 選擇是否參與社群功能
  • 管理通知提醒設定

6. 兒童隱私保護

本應用不針對 13 歲以下兒童設計。我們不會故意收集 13 歲以下兒童的個人資料。如果您發現有兒童向我們提供了個人資料，請立即聯絡我們，我們將採取措施刪除相關資料。

7. 資料保存期限

資料類型 保存期限 說明
個人記錄 永久（直到您刪除） 儲存在設備本地
錄音檔案 處理後立即刪除 雲端不保留
小樹洞貼文 有限期間 依系統運行需求
漂流瓶 有限期間 對話結束後清理
分析資料 依服務需求 匿名統計使用

8. Cookie 和追蹤技術

本應用不使用 Cookie。我們使用以下技術來改善服務：

  • Firebase Analytics：匿名收集使用統計
  • Firebase Crashlytics：收集崩潰報告
  • 本地儲存：儲存應用設定和偏好

9. 隱私政策變更

我們可能會不定期更新本隱私政策。任何變更都會在本頁面公布，並更新「最後更新日期」。重大變更時，我們會透過應用內通知告知您。建議您定期查看本隱私政策。
