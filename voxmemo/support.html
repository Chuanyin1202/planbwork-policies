關於 Voxmemo

Voxmemo 是一款智慧語音備忘錄應用程式，讓您能夠透過語音快速記錄待辦事項和支出收入。應用程式提供 AI 智能分類、時間解析，以及本地化資料儲存。

主要功能

  • 語音錄製：錄製語音備忘錄並自動轉換為文字
  • 智能分類：AI 自動識別待辦事項類型（活動/清單）
  • 時間解析：自動解析語音中的時間和地點資訊
  • 記帳功能：快速記錄支出和收入
  • 本地儲存：所有資料安全儲存在您的設備中

常見問題

應用程式無法錄音？

請確認您已授權 Voxmemo 使用麥克風權限：

  • iOS：設定 → 隱私權與安全性 → 麥克風 → Voxmemo
  • Android：設定 → 應用程式 → Voxmemo → 權限 → 麥克風

語音轉文字不準確？

語音轉文字使用 OpenAI Whisper API 處理。為獲得最佳效果，請：

  • 在安靜的環境中錄音
  • 說話清晰，語速適中
  • 避免過多背景噪音

待辦事項分類錯誤？

AI 智能分類基於語意分析，可能偶爾出現誤判。您可以：

  • 手動修改待辦事項的類型
  • 編輯待辦事項內容以更準確表達意圖

記帳記錄如何備份？

目前 Voxmemo 將所有資料儲存在您的設備本地。刪除應用程式將同時刪除所有資料，請在刪除前確認已完成必要的備份。

應用程式閃退或運行緩慢？

請嘗試以下解決方法：

  • 重新啟動應用程式
  • 檢查是否有應用程式更新
  • 確保設備有足夠的儲存空間
  • 重新啟動設備
